# FeigenbaumUtil.getCyclicPermFromLambda — Function.

getCyclicPermFromLambda(f,period,lam,x0)

Given a map and parameter value corresponding to a cycle of period 'period' returns the associated cyclic permutation

Example

julia> f,fc,maxloc = selectmap("Log");
julia> getCyclicPermFromLambda(f,3, 0.9579685138208287, maxloc)
[2 3 1]
